我经常一个人背着器材在野外走很久，然后什么都不拍。

听起来很浪费对吧？但对风光摄影来说，**等待本身就是拍摄的一部分。**

这不是鸡汤，是我真的这么认为。**如果这个宇宙的结局是注定的，是消亡，是归于死寂，那或许这更说明只有过程才是最最重要的。** 拍照不是为了那张照片，拍照是为了那个你在山顶等光的过程。

今天说几个我实践了很久的方法，不一定对，但都是真实的经验。

## 1. 光是唯一重要的东西

器材党最大的谎言：画质决定一切。

{% img /img/photos/IMG_4924.jpg '"黄金时刻" "日落摄影"' %}

这张照片是在日落前20分钟拍的。你看那个金色，那个橘红色——参数？Sony A7 IV，f/8，ISO 100，1/60秒。这些信息一点都不重要。

重要的是那个时间、那个地点、那片天空恰好出现了这样的颜色。**我能做的只是在正确的时间出现在正确的地点，然后按下快门。**

怎么做到？三个字：**看天气。**

不是为了发朋友圈那种看法，是真正去读云层、读光线的方向、读能见度。我手机里装了三四个天气App，每天看云图跟看股票一样认真。

极端吗？**风光摄影本身就是一件需要付出极端代价的事。** 我在零下五度的海边等过日出，手指完全没了知觉。也在暴雨天蹲过，最后相机进水了，照片也没拍成。

你问我值不值？**值不重要。重要的是你愿不愿意为那个瞬间付出过程。**

## 2. 画面的层次是走出来的

一张照片看起来很「平」，通常只有一个原因：**你站着拍。**

{% img /img/photos/IMG_4933.jpg '"前景构图层次" "风光摄影构图"' %}

这张照片我几乎是趴在地上拍的。取景框离地面大概20厘米，前景的草变成了虚化的色块，视线沿着地面延伸到远方的山。

**如果当时我站着拍，这就是一张完全无趣的照片。** 蹲下来，画面立刻有了前中后三层。

做内容也是一样的逻辑。一张好封面和一张普通封面的区别，往往就是多花了30分钟去迭代。**极致质量——这是我做任何事的原则。** 你可以不认同，但你不能否认用户能感觉得到。

## 3. 引导线不是找出来的，是走出来的

很多人拿着相机到处找「引导线」，这其实搞反了。

{% img /img/photos/IMG_4946.jpg '"引导线引导视觉" "摄影构图技巧"' %}

好的引导线不在那里等你发现，**它是你调整位置后「创造」出来的。** 往左走三步、蹲低一点、换个焦段——同样的场景，引导线就出来了。

在影视飓风，我经常跟团队说一句话：**结构比人可靠。** 与其寄希望于某个人的灵光一现，不如建立一个系统，让好内容可以被稳定地生产出来。

构图也是一样：**不要等灵感，要建立自己的工作流。** 每次到一个新机位，按顺序做这三件事：

1. 先绕着场景走一圈，找到最好的光线方向
2. 蹲下、站直、后退、前进，每个位置都试一遍
3. 确定前景、主体、背景三层的关系

我敢保证，做完这三步，你拍的照片已经超过了90%的游客照。

## 4. 框架构图是一种「压迫感」

你知道框架构图为什么好看吗？不是说有个框在画面里就叫构图了。

{% img /img/photos/IMG_4956.jpg '"框架构图" "摄影构图技巧"' %}

**好的框架构图，是让观众产生一种「偷窥感」——仿佛你在透过一个隐秘的视角，窥探另一个世界。**

这种压迫感在电影里经常用到。诺兰就特别喜欢用框架来创造沉浸感——你通过那个框看过去，感觉自己也进入了那个空间。

摄影也是一样。树枝、门框、岩石的缝隙——把它们放在画面边缘，观众的目光会被自然地「挤压」到画面的核心区域。**这不是技巧，是心理学。**

## 5. 慢门 = 让时间显形

你知道流水肉眼看起来是什么样的吗——液态的，流动的。但如果你用30秒快门去看——

{% img /img/photos/IMG_4964.jpg '"慢门拍水" "长曝光摄影"' %}

它就变成了丝绸、薄雾、或者一片虚幻的白。

**慢门本质上不是一种拍摄方法，它是一种「让时间显形」的方式。** 我们肉眼看不到时间，但相机可以——它把几秒钟压缩到了一个画面里。

拍摄慢门的核心，不是你买了多贵的ND镜，是你有没有三脚架——以及你有没有那个耐心。

我跟你说一个数据：拍一张好的慢门照片，从架好三脚架到出片，平均需要45分钟。这45分钟里，你什么都做不了，只能看着水流、等着风停、祈祷不要有人闯进画面。

**但这45分钟，才是这张照片真正有价值的部分。** 没有这45分钟的过程，那张照片只是一个技术展示。

{% img /img/photos/IMG_4950.jpg '"风光摄影作品" "摄影作品"' %}

## 关于风光摄影，我最后说几句

可能你发现了，这篇文章没有太多参数、没有太多「按这个按钮」。因为我觉得那些东西，说明书上都有，不需要我来讲。

我想讲的是：**你为什么拍照？**

你花了两万块买相机，花了一万块买机票去川西，在海拔四千米的地方等了两小时的日落——最后拍到的照片，可能还不如别人的手机随手拍。

**但你经历的那些过程——缺氧、寒冷、等待、失望、偶尔的惊喜——这些才是摄影真正给你的东西。**

这个宇宙的结局是注定的，所有的照片最终都会消失，所有的数据最终都会归零。

**但那个过程，是你真实拥有过的。**

无限进步。
