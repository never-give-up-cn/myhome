构图是摄影的基石。好的构图能让一张普通的照片变得引人入胜。今天分享几个常用的风光摄影构图技巧。

## 1. 三分法

这是最基本也最常用的构图法则。把画面用两条竖线和两条横线分成九宫格，将主体放在交叉点上：

```
+-------+-------+-------+
|       |       |       |
|   ·   |   ·   |   ·   |
|       |       |       |
+-------+-------+-------+
|       |       |       |
|   ·   |   ·   |   ·   |
|       |       |       |
+-------+-------+-------+
```

- 拍摄海景时，把海平线放在上1/3或下1/3处
- 拍摄独立树木时，把它放在左侧或右侧1/3处

## 2. 引导线

利用画面中的线条引导观众视线：

- **道路** — 蜿蜒的公路、小路
- **河流** — 溪流、海岸线
- **建筑** — 桥梁、走廊、栅栏

引导线从画面前景延伸到远处，能创造出强烈的纵深感。

## 3. 前景构图

在画面中加入前景元素，增加层次感：

```
[前景] → [中景] → [远景]
 花朵      湖面      山峦
```

比如拍摄日出时，前景放几块礁石或一些野花，画面会立刻生动起来。

## 4. 框架构图

利用自然或人工框架框住主体：

- 树枝和树叶
- 窗户和拱门
- 岩石的缝隙

框架不仅能突出主体，还能增加画面的故事感。

## 5. 对称与平衡

水面倒影是最常见的对称构图：

```
天空 ⬆️
—————————— 水平线
倒影 ⬇️
```

适合拍摄：湖泊、雨后积水、建筑倒影。

## 小贴士

> **Rule of thirds is meant to be broken.**
> 三分法就是用来被打破的。

法则只是起点。当你熟悉了这些规则之后，大胆去尝试突破它们，找到你自己的风格。

## 总结

| 技巧 | 适用场景 | 难度 |
|------|---------|------|
| 三分法 | 大多数场景 | ⭐ |
| 引导线 | 风光、街道 | ⭐⭐ |
| 前景构图 | 风光、花鸟 | ⭐⭐ |
| 框架构图 | 人文、建筑 | ⭐⭐⭐ |
| 对称构图 | 倒影、建筑 | ⭐ |

下次出去拍照的时候，试着有意识地运用这些构图技巧，你会发现照片质量立刻提升一个档次！🚀
