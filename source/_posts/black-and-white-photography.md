你有没有想过一个问题——**颜色对一张照片来说，到底是加分项还是干扰项？**

我第一次认真思考这件事，是在看一张黑白照片的时候。那是一张很老的街拍，画面里只有一个路人走过水洼。没有颜色，没有 fancy 的特效，甚至连画质都算不上好。

但我盯着那张照片看了很久。

因为它让我感觉——**这就是那一刻的样子。没有滤镜，没有修饰，光、影、人的姿态，全部赤裸裸地摆在你面前。**

彩色照片记录的是世界的表象，而黑白照片记录的，是世界的结构。

{% img /img/photos/P1010022.JPG '"黑白摄影的开篇" "黑白影像"' %}

## 颜色是糖衣，光影才是真相

我做了这么多年影像相关的内容，越来越觉得：**颜色有时候是一种逃避。**

什么意思？当你构图不够好的时候，可以用色彩来分散注意力。当你光影不够丰富的时候，可以用色调来弥补。色彩就像一层糖衣——它让照片看起来「好看」，但也掩盖了很多本质的问题。

黑白摄影最「恐怖」的地方就在这里：**它把所有可以藏拙的东西都拿走了。**

剩下的只有三样东西：**光、影、构图。**

{% img /img/photos/P1010026.JPG '"黑白摄影中的光影层次" "光影"' %}

你看这张照片。没有颜色帮你引导视线，你只能靠灰度之间的过渡来阅读画面。亮部在哪里？暗部在哪里？灰度渐变是否平滑？——这些问题在彩色照片里你几乎不会去想，但在黑白照片里，它们就是全部。

## 灰度就是节奏

我做影视飓风这么多年，一直在用 HKRR 来检验内容——Happiness、Knowledge、Resonance、Rhythm。

黑白摄影特别有意思的一点是，**它天然就把 Rhythm（节奏）提到了最前面。** 因为颜色消失之后，画面里唯一能制造节奏感的东西，就是灰度之间那种微妙的排列——从黑到灰再到白的过渡，像是钢琴键上从低音到高音的排列。

{% img /img/photos/P1010032.JPG '"灰度层次" "黑白摄影构图"' %}

你看这张照片里的灰度分布。它不是均匀的——有一段区域是深灰，有一段跳到了亮白，然后又沉下去。这种黑白的起伏，就像音乐里的旋律线。

我拍黑白的时候经常想：**如果这张照片转成黑白的那个瞬间不好看，那它本身就不好看。** 跟颜色没有关系。

## 一张好照片，跟颜色无关

这听起来可能有点绝对，但我是认真的。

{% img /img/photos/P1010040.JPG '"黑白中的人文气息" "街拍黑白"' %}

你觉得打动你的，是那个颜色，还是那个人、那个姿态、那个瞬间？

我敢说，90% 让你印象深刻的照片，你把颜色抽掉之后依然是好照片。而那些靠颜色撑场面的照片，一旦变成黑白就什么都没了。

## 怎么拍好黑白

说点实用的。

### 1. 在取景框里就「转黑白」

不要拍完了彩色的再去后期转黑白——**在按下快门前就要用黑白的方式去思考。**
	
我在取景的时候，会刻意忽略颜色，只看这几个东西：
- 光从哪里来，照到了哪里
- 阴影的形状和位置
- 线条的走向
- 灰度之间的层次

{% img /img/photos/P1010048.JPG '"黑白构图练习" "黑白摄影技巧"' %}

### 2. 找反差

黑白靠反差吃饭。大面积的亮部旁边放一块暗部，观众的视线会瞬间被吸引过去。

没有反差的黑白照片，就是一团灰色的雾。

### 3. 纹理是黑白独有的语言

彩色照片里纹理常常被颜色淹没，但黑白照片里——**纹理就是主角。**

{% img /img/photos/P1010056.JPG '"黑白纹理细节" "黑白摄影细节"' %}

墙上的裂纹、水面的波纹、树叶的脉络——这些在彩色照片里不起眼的东西，在黑白照片里会变得非常有力。因为去掉了颜色，你的眼睛会不由自主地去找那些触觉层面的信息。

### 4. 曝光要精准

黑白对曝光的要求比彩色更严格。彩色照片里，你过曝一档肤色还能看，但黑白照片里过曝——**那片死白是拉不回来的。**

拍黑白，我习惯欠曝半档到一档。暗部可以提亮，但亮部死了就是死了。

{% img /img/photos/P1010064.JPG '"黑白中的细节保留" "黑白曝光技巧"' %}

## 从黑白照片，到宇宙的本质

你有没有发现，我们提到「经典」的照片时，脑子里浮现的大部分是黑白的？

布列松、亚当斯、罗伯特·卡帕、寇德卡——那些跨越时代的影像，绝大部分是黑白的。这不是偶然。

**因为去掉颜色之后，一张照片回到了它最本质的状态：光和时间的记录。**

{% img /img/photos/P1010072.JPG '"黑白中的光与影" "黑白摄影"' %}

我看这张照片的时候在想：人类对光的感知本来就是黑白的。我们的视杆细胞比视锥细胞敏感得多——在暗光环境下，我们看到的本来就是灰度世界。

**所以黑白不是一种风格，它是一种更原始的观看方式。**

这个宇宙本身也没有颜色。颜色只是不同波长的电磁波在我们大脑里的映射。去掉颜色，你反而离真实更近了一步。

这可能就是我喜欢黑白摄影的原因——不是因为它「高级」，而是因为它**真实**。

而真实，永远是值得追求的东西。

无限进步。
