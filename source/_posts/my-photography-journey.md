前几天整理硬盘的时候，看到自己这些年的照片文件夹——从2017年到现在，零零散散攒了快 **16万张** 照片。

我盯着那个数字看了一会儿。

16万张。按快门16万次。这里面有拍的好的，有拍的一般的，有大量完全不知道在拍什么的。但它们串起来，就是一个很普通的故事——**一个业余摄影爱好者，从高中拿着手机乱拍，到端着一台全画幅松下认真构图的七年。**

{% img /img/photos/IMG_20181129_231155.jpg '"2018年冬天，手机拍的" "手机摄影"' %}

## 2017年：一切都始于一部手机

跟很多人的故事一样，我摄影的起点不是什么伟大的理想——**就是一部手机。**

那时候还在上高中，哪懂什么光圈快门构图。就是觉得有些画面挺好看的，想留下来。学校傍晚的走廊、同桌趴在桌上睡觉的样子、食堂阿姨多打了一勺菜——什么都拍。

现在回头看那些照片，构图一塌糊涂，画质放到今天根本没法看。

**但我一次都没舍得删。**

{% img /img/photos/IMG_20181130_100652.jpg '"2018年，手机记录的日常' "手机摄影"' %}

因为我后来慢慢明白了一件事：**你最开始拍的那些「烂片」，恰恰是最珍贵的东西。** 它们记录的不是技术，是那个还没有被器材、参数、画质这些概念污染的、最纯粹的观看欲望。

## 2019年：第一台相机——佳能M100

高中毕业之后，我买了第一台真正的相机——**佳能M100**。

那时候选它的理由很单纯：好看、便宜、能换镜头。 APS-C画幅，2400万像素，在今天看来很入门的配置。但当时拿到手的那一刻，我觉得自己专业了——终于可以用取景器看世界了。

{% img /img/photos/IMG_20181201_103517.jpg '"那段时间拍的" "相机摄影入门"' %}

M100陪了我很长一段时间。我带着它拍过很多东西——周末出去玩、家里的猫、路边的花。它的对焦不算快，高感也不算好，但它让我学会了三样最重要的东西：

- 光圈是什么、怎么用
- 光线决定一切
- **拍照本身就是一件值得开心的事**

> **器材从来不是限制你拍照的理由。** 那些拿着入门机也能拍出好照片的人，不是因为他们技术好，是因为他们根本没去想器材的事，他们只是单纯地在拍。

## 2023年：换到佳能M3

后来我换了佳能M3。比M100大了一点，操控专业了一点，但本质上还是APS-C。那时候我其实已经在看全画幅了，但因为预算原因，还是在半画幅上多待了几年。

现在回想起来，**这段「够不着」的时间，反而是进步最快的。** 因为你知道自己的器材有限制，所以你会在构图和用光上更动脑子。你不会觉得"先拍下来再说，后期裁一下就好了"——因为像素就那么点，裁不起。

{% img /img/photos/IMG_4924.jpg '"后来拍的风景" "风光摄影"' %}

## 到现在：松下S5M2

终于，我换到了全画幅——**松下S5M2**。

第一次拿到它的时候，确实能感受到差距。高感好了很多，动态范围大了很多，拍夜景的时候终于不用那么挣扎了。

{% img /img/photos/P1010022.JPG '"松下S5M2拍的" "全画幅摄影"' %}

但有趣的是，**第一张让我自己满意的照片，并不是全画幅拍的。**

它是一张用手机拍的、画质很差的、构图也没什么讲究的照片。但我看到它的时候，能想起那天的心情、温度和气味。

{% img /img/photos/IMG_20181130_170915.jpg '"那些日子" "生活记录"' %}

## 从广角到长焦

镜头也攒了不少。从套机头开始，到后来的广角、标准变焦、长焦——

- 广角让我学会了怎么拍出气势
- 标变是最常用的，它教会我构图靠走
- 长焦让我知道，有些东西你不需要靠近，远远地看着也挺好

每颗镜头都改变了我看世界的方式。

## 165,792次快门

这篇文章的最后，我想告诉你一个数字——

我统计了一下 `E:\照片` 这个文件夹，里面一共有 **165,792 张** 照片和图片。

从2017年高中那部手机开始，到2019年的佳能M100，到2023年的佳能M3，到现在的松下S5M2。从广角到长焦，从白天拍到晚上，从乱拍到慢慢有了一点自己的想法。

165,792 次按下快门。

大部分都是废片，这是实话。但那些偶尔拍好的、偶然碰到的美好瞬间——**它们让前面几万次的"浪费"都变得值得了。**

{% img /img/photos/IMG_4950.jpg '"继续拍下去" "摄影旅程"' %}

## 写在最后

我不是什么专业的摄影师，到现在也不敢说自己拍得多好。

我只是一个跟很多人一样的业余爱好者——**喜欢拍照，享受按下快门那个瞬间的感觉。**

器材从手机换到全画幅，镜头从广角买到长焦，照片从几万张拍到十几万张。变了很多，但有一件事从来没变过：

**我还是会因为看到一片好看的光影，就忍不住停下来。**

不管手里拿的是手机还是全画幅。

无限进步。
