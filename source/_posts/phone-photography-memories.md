我最近翻了一下手机相册，看到一堆2018年的老照片。像素不高，构图随意，有些甚至还是虚的。

但我看了很久。

因为这些照片让我想起了一个事实：**那些用手机随手拍下的照片，才是我们回头看时最想看到的。**

{% img /img/photos/IMG_20181129_231155.jpg '"深夜的随手拍" "手机摄影"' %}

## 我的第一台相机，其实是手机

很多人知道我拍照片、聊器材，但你可能不知道——我最开始根本没有什么好相机。

跟大部分人一样，**我的第一台相机就是手机。**

那时候也不懂什么光圈快门，就是看到想记录的，拿出来拍。拍的不好看？没关系，反正手机里随时可以再拍。

现在想想，那种「随便拍」的状态，反而是最珍贵的。

{% img /img/photos/IMG_20181130_100652.jpg '"早晨的记录" "手机拍照"' %}

因为你不加思考地按下快门时，拍到的往往是最真实的东西。没有构图压力，没有「这张能不能发朋友圈」的包袱——你就是单纯地在记录。

## 手机和相机的本质区别

我用过很多相机，从入门单反到旗舰全画幅。但我必须说一件事：

**相机让你拍得更好，手机让你拍得更多。**

{% img /img/photos/IMG_20181130_170915.jpg '"下午的记录" "日常随拍"' %}

这两者之间的区别，比你想象的要大得多。

相机有门槛。你要带上它，要设置参数，要思考构图——这些都会让你在按下快门前多犹豫几秒。而这几秒，有时就会错过那个瞬间。

手机没有门槛。它就在你口袋里，你拿出来就拍。可能画质一般，可能噪点很多，但那个瞬间，你抓住了。

{% img /img/photos/IMG_20181201_103517.jpg '"另一天的记录" "手机生活摄影"' %}

## 2018年的像素，2026年的回忆

这篇文章里配的这些照片，都是2018年拍的。那时候的手机画质放到今天来看，确实一般。暗光下噪点很多，细节也不够丰富。

但你看，**我根本不在乎这些。**

我看着这些照片的时候，想的不是「这个噪点有点多」——我想的是那天我去了哪里、跟谁在一起、那时候在做什么。照片只是钥匙，它打开的是记忆。

{% img /img/photos/IMG_20181201_182443.jpg '"傍晚的记录" "手机摄影"' %}

{% img /img/photos/IMG_20181202_104151.jpg '"又一天的记录" "生活记录摄影"' %}

我经常说一句话：最好的照片，往往没什么特别的。

我当时按下快门的时候，肯定没想过这些照片有一天会出现在一篇文章里。就是觉得那个瞬间值得拍下来，仅此而已。

但就是这些「没什么特别的」瞬间，拼凑起了我们的生活。

## 所以我想说的是

你可能正在纠结要不要买一台好的相机，或者觉得自己的手机拍照不够好。

我的建议很简单：**先把你手机里的照片拍好。**

不是说技术层面的「好」——是多拍。

{% img /img/photos/IMG_20181213_094438.jpg '"再一个日常" "手机随拍"' %}

你不需要等到有了一台好相机才开始记录。你手里的手机已经够好了。它最好的地方不是画质，而是**它永远在你身边。**

十年后，你不会后悔那些用手机拍下的普通日常。你只会后悔那些你按下快门的瞬间。

{% img /img/photos/IMG_20181214_075105.jpg '"早晨的记录" "手机摄影记录"' %}

## 写在最后

这篇文章里的照片拍于2018年。那时候我不知道后来会写文章、做内容、跟大家聊摄影。我就只是拿着手机，拍自己想拍的。

现在回头看，那些照片的像素确实不怎么样。

**但它们是真实的。**

而真实，才是摄影唯一重要的东西。

拿起你的手机，去拍吧。

无限进步。
